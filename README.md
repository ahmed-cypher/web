   `
   example.com
   testsite.org
   `
2. أنشئ ملف نصي يحتوي على محتوى تجريبي (مثلاً: test.txt).
   `
   Hello Vulnerability Test
   `

   `bash
   python3 tester.py
   `
4. أدخل مسار الملفات عندما يطلب منك.
5. بعد الانتهاء، ستجد النتائج في ملف vulnerable_sites.txt.
